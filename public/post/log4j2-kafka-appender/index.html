Log4j2 Kafka Appender

May 5, 2018 65 words 1 min read

Table of Contents

        • Stack

This example demonstrates how to send logs to Kafka using log4j2 KafkaAppender.

Stack

  • Java 8
  • Kafka 1.1.0
  • Maven 3.5

The oneLiner script was tested on Ubuntu 16.04 with docker & docker-compose.

  • git clone https://github.com/tecris/blog-samples
  • cd log4j2-kafka
  • $ ./oneLiner.sh

log4j2.xml configuration:

<Appenders>
  <Kafka name="Kafka" topic="${sys:kafkaTopic}">
    <PatternLayout pattern="[%-5level] %d{yyyy-MM-dd HH:mm:ss.SSS} [%t] %c{1} - %msg%n"/>
    <Property name="bootstrap.servers">${sys:kafkaUrl}</Property>
  </Kafka>
  ...
</Appenders>
